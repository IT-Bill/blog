## 前言

作为一个新入手 Mac 的用户，快速上手是非常重要的。本文将介绍一些基本的操作和技巧。

## 基本操作

### 触控板手势

Mac 的触控板非常强大，支持多种手势操作：

- 双指滚动
- 双指缩放
- 三指拖动
- 四指切换桌面

### 快捷键

常用快捷键：

- `Command + C` 复制
- `Command + V` 粘贴
- `Command + Space` 聚焦搜索
- `Command + Tab` 切换应用

## 推荐软件

1. Homebrew - 包管理器
2. iTerm2 - 终端增强
3. VS Code - 代码编辑器
4. Rectangle - 窗口管理
